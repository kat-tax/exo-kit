# Video

> Displays video content via [rnv](https://react-native-video.github.io/react-native-video) and web via [shaka](https://github.com/shaka-project/shaka-player)

<Video
  muted
  controls
  source={{
    uri: 'https://get.ult.dev/samples/bunny.mp4',
  }}
/>

```tsx twoslash
import React from 'react';
// ---cut---

// @log: ↓ Try the example

  source={{

  }}

```

## Dependencies

:::warning
You must install [React Native Video](https://react-native-video.github.io/react-native-video)

```sh
npm i react-native-video
```
:::
