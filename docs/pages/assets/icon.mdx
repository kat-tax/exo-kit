## Dependencies

:::warning
You must install [React Native SVG](https://github.com/software-mansion/react-native-svg)

```sh
npm i react-native-svg
```

For native, you must modify your **babel.config.js** file:

```js
{
  plugins: ['react-exo/babel-plugin-iconify']
}
```
:::

## Technology

| Platform | Library | Support | Notes |
| -------- | ---------- | :--------: | :---- |
| Web | [@iconify/react](https://iconify.design/docs/icon-components/react) | ✅ | Loads icons on demand |
| Native | *custom* | ✅ | Used icons cached in bundle |
