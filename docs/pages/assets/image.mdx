## Dependencies

:::warning
For Android & iOS support you must install [Faster Image](https://github.com/candlefinance/faster-image)

```sh
npm i @candlefinance/faster-image
```
:::

## Technology

| Platform | Library | Support | Notes |
| -------- | ---------- | :--------: | ----- |
| Web | *custom* | ✅ | Default platform caching |
| Android | [faster-image](https://github.com/candlefinance/faster-image) | ✅ | - |
| iOS | [faster-image](https://github.com/candlefinance/faster-image) | ✅ | - |
| MacOS | *custom* | ✅ | Default platform caching |
| Windows | *custom* | ✅ | Default platform caching |
