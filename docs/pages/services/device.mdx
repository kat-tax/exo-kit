# Device

> A service to manage and query device instruments

```ts twoslash
// @noErrors
import React from 'react';
// ---cut---
// @log: ↓ Import the device service
import {Device} from 'react-exo/device'; // [!code focus]

// @log: ↓ Share EXO with a friend :]
Device.share('https://exo.ult.dev', 'EXO is amazing!'); // [!code focus]

// @log: ↓ Query online status
const online = await Device.isOnline(); // [!code focus]

// @log: ↓ Query device locale
const locale = Device.getLocale(); // [!code focus]
​
```

## Dependencies

:::warning
For native support you must install [RN Net Info](https://github.com/react-native-netinfo/react-native-netinfo)

```sh
npm i @react-native-community/netinfo
```
:::
