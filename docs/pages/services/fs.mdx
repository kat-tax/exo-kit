# File System

> A service for working with the file system.

```ts twoslash
// ---cut---
// @log: ↓ Import the File System service
import {FS} from 'react-exo/fs';

// @log: ↓ Initialize the file system
const fs = await FS.init();​
​
```

## Dependencies

:::warning
For native support you must install [react-native-file-access](https://github.com/alpha0010/react-native-file-access)

```sh
npm i react-native-file-access
```
:::
